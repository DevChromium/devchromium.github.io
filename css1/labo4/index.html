Opdracht 3

1. Waarom staat de tekst in de blokken horizontaal gecentreerd?

Op elke div wordt er een text-align: center; toegevoegd.

2. Ga na dat een property vertical-align: center; geen effect heeft. Waarom eigenlijk niet?

"center" is geen mogelijke optie voor een vertical-align property.

3. Ga na dat vertical-align: middle; er niet voor zorgt dat de tekst verticaal gecentreerd wordt. Wat doet dit dan wel?

vertical-align zorgt er voor dat de tekst op de lijn zelf gepositioneerd wordt. Dit werkt niet op de parent element..

4. Wat zijn de andere mogelijkheden om text te centreren?

.center {
  text-align: center;
  line-height: 100px;
  height: 100px;
}
/* Eerder zou ik dit gebruiken */
.center {
  display: flex;
  align-items: center;
  justify-content: center;
}

5. In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang?

Omdat de height van de area class overschreden wordt met de height value van de tall class.
